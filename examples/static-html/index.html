AI Chat Widget Demo

The widget is loaded from the jsDelivr CDN — no build step or npm install needed. Click the floating button in the corner to open the chat panel.

Backend required: the widget sends POST /ai-chat/custom to the same origin by default. Point it at your server with backend-url and path attributes. It expects { "message": "..." } or { "response": "..." } in the response body.

See ai-chat-toolkit-server for a ready-made Express backend that works with this widget.
