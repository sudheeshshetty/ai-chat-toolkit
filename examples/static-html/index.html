AI Chat Widget Demo

This page loads the locally built browser bundle from packages/widget/dist/widget.global.js. Click the floating button in the corner to open the chat panel.

Backend required: ensure a server is running that accepts POST /ai-chat/custom on this origin (or set backend-url and path on the element). The widget sends JSON with message and history, and expects { "message": "..." } or { "response": "..." } in return.
